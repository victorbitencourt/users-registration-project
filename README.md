# Users Registration Project

This project was developed during the "Angular: O Guia Definitivo das Principais Funcionalidades" course on Udemy.

## Technologies and tools used

- Angular 16
- TypeScript
- Angular Material
- Angular CDK
- RxJS
- HTML5
- SCSS
- Zxcvbn

## Features

- List of users
- Editing information
- Complete validation of forms
- Password confirmation
- Password strength indicator
- Brazilian state selection
- Music genre selection
- Date conversion and formatting
- Comparison of data before/after editing using dialog

## Applied concepts

- Componentization
- Services
- Dependency Injection
- Interfaces
- Types
- Pipes
- Directives
- Template-driven Forms
- Custom validations
- Layered organization
